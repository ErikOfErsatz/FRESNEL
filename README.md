FREE RESOURCE FOR EDITING STATES, NABBED-ITEMS, ENERGY & LOCATIONS

Free Save Data Editor for USERLANDS' Eternity Egg

Open the app by double-clicking the "fresnel_20260105.jar" file.

1. Load your PlayerState.json file using the "Select PlayerState.json" button.
2. Add checkpoints on the left with the selector next to the "Add" button.
3. Set your current checkpoint with the selector below the add selector.
4. Add items on the right with the selector next to the "Add" button.
5. Remove items and checkpoints by selecting them in the list view and pressing Delete or Backspace.
6. Add or remove a single item by selecting it in the list view and pressing Left (-1) or Right (+1).
7. Revert all changes with the "Revert" button.
8. Create a new, modified PlayerState.json with the "Update" button.
9. Your new PlayerState.json file will appear in the same directory as the application.
10. Replace your old PlayerState.json with the updated file and load the game
11. Have fun!
